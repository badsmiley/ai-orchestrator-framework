# AI Orchestrator Framework

**Open source PMBOK для работы с AI-агентами и многоуровневыми системами.**

## Концепция

Этот фреймворк описывает, как выстроить работу с AI-агентами по принципу "оркестратор + департаменты" — без единой точки отказа, с параллельным выполнением задач и чётким протоколом коммуникации.

## Структура

```
ai-orchestrator-framework/
├── principles/          # Базовые принципы и философия
│   └── OPERATING_MODEL.md
├── templates/           # Шаблоны для проектов и агентов
│   └── project-template/
├── runbooks/            # How-to гайды
│   └── session-spawn.md
├── docs/                # Документация
│   └── agent-levels.md
└── README.md
```

## Уровни системы

| Уровень | Роль | Ответственность |
|---------|------|-----------------|
| **L0** | Orchestrator (главный чат) | Цели, приоритеты, декомпозиция, управление департаментами |
| **L1** | Department (субагент) | Отдельный проект/домен, репортит L0 |
| **L2** | Worker (опционально) | Исполнительные задачи внутри департамента |
